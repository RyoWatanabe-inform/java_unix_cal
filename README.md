# ソースコードの実行方法
※推奨OS：Window11  
①ソースを実行環境にダウンロードします。（JAVAが実行できるフォルダに格納）  
②コンソールで「bin」に移動します。  
③以下のコマンドでプログラムを実行します。  
　「java cal_command.Cal」
